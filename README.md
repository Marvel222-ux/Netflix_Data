# Netflix_Data

## Tools Used
- Microsoft Excel/Metabase
- Power BI
- Visualization
- Pivot Tables
- Charts & Graphs
- Data Cleaning Techniques

## Dataset Description
The dataset contains:
- Show ID
- Type
- Title
- Director
- Cast
- Country
- Date_Added
- Release_Year
- Rating
- Duration
- List_In
- Description

## Data Cleaning Process
- Handled missing values
- Standardized data format
- Oranized data for analysis

## Dashboard Features
- Count of Title by Year
- Rating by Type
- Top 10 Director
- Top 10 Cast
- Total show by Type
- Total of Titles by Country
